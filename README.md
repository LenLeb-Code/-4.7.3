# Прототип системы ИИ-скоринга кандидатов

Система для автоматизации первичного скрининга соискателей на основе анализа их резюме, финансовых показателей и цифрового следа.

## Как это работает
Система использует гибридный подход:
1. **NLP (BERT)** — анализирует текст резюме и извлекает ключевые навыки.
2. **Gradient Boosting (CatBoost)** — оценивает благонадежность на основе кредитного рейтинга и транзакционной активности.

## Пример работы (Dashboard)

| Кандидат | Скор (ИИ) | Ключевые факторы (SHAP) |
| :--- | :---: | :--- |
| **Иванов И.И.** | **89/100** | + 5 лет опыта в IT; + Высокий кредит-балл. |
| **Петров П.П.** | **42/100** | - Наличие просрочек по кредитам; - Траты в МФО. |

## Стек технологий
* **Python**: Обработка данных и ML-логика.
* **PostgreSQL**: Хранение структурированных анкетных данных.
* **BERT**: Модель для анализа текстов.
